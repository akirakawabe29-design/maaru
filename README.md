# MAARU

沖縄で挑戦する企業と、沖縄を発信するクリエイターをつなぐPRサービス「MAARU」のリブランディングWebサイトです。

## Preview

```bash
python3 -m http.server 4173
```

ブラウザで `http://localhost:4173` を開いて確認できます。

## Structure

- `index.html` - TOPページ
- `styles.css` - レイアウト・デザイン・レスポンシブ対応
- `script.js` - ナビゲーション・表示アニメーション
- `assets/images` - Webサイト用画像
- `brand` - ブランドガイドライン・ロゴデータ
- `scripts` - ロゴ変換・比較用スクリプト
